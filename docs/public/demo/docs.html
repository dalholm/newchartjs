Documentation

Components, styling, theming, and API reference for NewChart JS.

Getting Started

Installation, basic usage, and config structure

Include the library

<script src="dist/newchart.min.js"></script>

Create a chart

const chart = NewChart.create('#my-chart', {
  type: 'bar',
  data: {
    labels: ['Jan', 'Feb', 'Mar', 'Apr'],
    datasets: [{
      label: 'Revenue',
      values: [12000, 19000, 15000, 22000]
    }]
  }
});

The container element must have a defined width and height (CSS or inline). Minimum rendered size is 300×300 px.

Config structure

KeyPurpose
typeChart type: bar, line, area, pie, gauge, sparkline
data{ labels: [...], datasets: [{ label, values, color }] }
styleVisual appearance — colors, fonts, sizes, animation
optionsBehavioral settings — legend, tooltip, axes, stacking

Update & destroy

// Reactive partial update
chart.update({ data: { datasets: [{ values: [15000, 21000, 18000, 25000] }] } });

// Clean up
chart.destroy();

Renderer

SVG by default. Switches to Canvas automatically when data exceeds 5 000 points. Force with options.renderer: 'canvas' or 'svg'.

Export

const pngUrl = await chart.toPNG();   // Promise<string> data URL
const svgStr  = chart.toSVG();        // SVG markup string

KPICard

DataTable

Auto-generated data table with bidirectional hover sync

NewChart.create('#chart', {
  type: 'bar',
  data: { ... },
  options: {
    table: {
      enabled: true,
      viewMode: 'split',   // 'chart' | 'table' | 'split'
      maxHeight: 200
    }
  }
});
View modeDescription
'chart'Chart only, table hidden
'table'Table only, chart hidden
'split'Chart on top, scrollable table below

Hover sync is automatic: hovering a table row highlights the chart, and vice versa.

{ style: { animation: { duration: 0 } } }

// CSS
.no-anim { --nc-animation-duration: 0; }

// Respect user preference
@media (prefers-reduced-motion: reduce) {
  .chart-container { --nc-animation-duration: 0; }
}

Legend

Click a legend item to toggle series visibility. Datasets with dash: true or ref: true show a "(ref)" tag. The legend is interactive by default.

Tooltip

Rich tooltips with header, color swatches, and YoY change footer are built automatically for BarChart and LineChart when multiple datasets are present.

Table

Any chart can display an auto-generated data table with options.table.enabled: true. Hover sync between chart and table is bidirectional.
