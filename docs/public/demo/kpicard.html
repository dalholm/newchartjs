KPI Card

Config-driven KPI cards with value, change badge, sparkline, progress bar, and status indicator. Built as a first-class component in NewChart — perfect for ERP dashboards.

Four cards with sparkline, change indicator, and progress bar — just like a real ERP dashboard

Budget Tracking with Progress

Status indicator (green/yellow/red) and progress bar based on target

Sparkline Variants

Line, area, and bar — different visual styles for trend data

Selectable Cards (click to activate)

onClick handler with active state — typical for metric selection in dashboards

Custom Formatting

Custom formatValue, prefix/suffix, and decimals

Minimal Cards

Reactive Update

Click "Simulate Update" to see the card update with new values

Usage

How to create a KPI card

const card = NewChart.kpiCard('#my-element', { label: 'Revenue', value: 38920000, previous: 32410000, suffix: ' kr', target: 42000000, sparkline: { values: [2850, 3120, 2960, 3480, 3210, 3650], color: '#4c6ef5', variant: 'area' // 'line', 'area', or 'bar' }, onClick: ({ config }) => console.log(config.label) }); // Update reactively card.update({ value: 41500000 });
