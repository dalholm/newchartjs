================================================================================
                            QUICK START GUIDE
================================================================================

1. INSTALL DEPENDENCIES:
   cd /sessions/dreamy-relaxed-euler/mnt/newstats
   npm install

2. BUILD PRODUCTION BUNDLES:
   npm run build

3. START DEVELOPMENT SERVER:
   npm run dev
   (Open browser to http://localhost:3000/demo/)

4. CREATE YOUR FIRST CHART:

   const chart = NewChart.create('#my-chart', {
     type: 'bar',
     data: {
       labels: ['Q1', 'Q2', 'Q3', 'Q4'],
       datasets: [{
         label: 'Revenue',
         values: [120, 200, 150, 300],
         color: '#4F46E5'
       }]
     }
   });

5. UPDATE CHART DATA:
   chart.update({
     data: {
       datasets: [{ values: [150, 250, 200, 350] }]
     }
   });

6. CLEANUP:
   chart.destroy();
